1. 두 개의 PC는 모두 TCP서버를 열 수 있으면서 서버와 클라이언트 역할을 동시에 하게 됨 
  1. 프로그램 실행 시에 Port를 설정해야 함.

2. (1) 요약
  사용자로부터 연결을 위한 포트 번호를 입력 받음
  입력된 포트 번호로 TCP 서버를 열어 대기
  해당 포트가 이미 사용 중인지 확인하기 위한 예외 처리를 수행.
    net 패키지를 사용하여 포트가 이미 사용 중인지 확인하고, 필요한 경우 다른 포트를 선택하도록 프로그램을 구성
    net.Listen 함수를 사용하여 포트를 열고 포트가 이미 사용 중인 경우 net.Error 타입의 에러를 처리하여 다른 포트를 선택하거나 오류 메시지를 사용자에게 표시
  다른 PC에서 연결을 수립하기 위해 대기.

3. 그 이후 하나의 PC에서 다른 PC로 연결 한다면
  1. 클라이언트-서버 형태로 변함
  2. 동시에 PC1은 PC2에 연결, PC2는 PC1에 연결 하는 형식
  이 방식을 P2P (Peer-to-Peer) 네트워크 모델에 해당하며, 양방향 통신을 가능하게 함, P2P 네트워크에서 각 PC는 서로 클라이언트로 연결하고 서로의 서버에 연결하므로 데이터를 양방향으로 교환
