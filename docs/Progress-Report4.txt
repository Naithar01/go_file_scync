1. 본인 서버에서 실행중인 서버에 연결이 안 되도록 
  1. Port 확인
    1. 입력 된 포트가 실행 중인 서버의 Port와 동일한지 확인
  
2. 서버 실행 이후 서버 연결 시에
  1. 서버 연결 성공 시에 => 연결을 받았을 때 서로 맞 물렸을 때 폴더 선택으로 이동하게
    ex: PC1은 서버 실행 동시에 클라이언트로부터 연결을 받은 상태이면 폴더 선택으로 이동
    1. 서버 실행 상태 State
    2. 서버 연결을 한 상태 State 

3. 이후에 서로 폴더 정보를 보내 View에 보여줄 수 있게

* 중간 정리
1. 서버 실행 시에 서버가 실행 됐다는 State = True [Y]
2. 클라이언트가 서버에 접속 하면 접속이 됐다는 State = True [Y]
3. 서버에서 클라이언트로부터 연결을 받으면 연결을 받았다는 State = True [Y]
